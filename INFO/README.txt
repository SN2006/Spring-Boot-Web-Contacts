В проэкте я решил использовать все нужные мне http методы и для этого
в html теге form я добавлял атрибут th:method, с указанием нужного мне метода.
Я его использовал только для тех методов, которые не поддерживаються обычными формами
(то есть PUT и DELETE).

Так же я решил поиграться с валидацией форм, для этого использовал Map<String, String> eroors,
который передавал в model, где с помощью атрибута th:if проверял содержит ли errors поле в своих ключах
и если да, то я выводил значение, которое соответствует этому ключу.

Во время разработки проэкта заметил, что Thymeleaf в Spring Boot сразу поддерживает кириллицу,
что не может не радовать)
